# Testing Infrastructure

This directory contains essential testing infrastructure for the SEU Injection Framework.

## Files

### `example_networks.py` - **CRITICAL - DO NOT REMOVE**
- **Purpose**: Provides test neural networks (NN, CNN, RNN) for integration testing
- **Coverage**: 100% - fully tested with comprehensive test suite
- **Used by**: Integration tests in `tests/integration/test_workflows.py`
- **Functions**:
  - `get_example_network()`: Creates test models with training data
  - `generate_moons_data()`: Creates synthetic datasets for testing
  - Model classes: `SimpleNN`, `SimpleCNN`, `SimpleRNN`

### `benchmark_results.jsonl`
- **Purpose**: Performance benchmark data from testing runs
- **Format**: JSON Lines format with timing and system information
- **Generated by**: Benchmark tests in `tests/benchmarks/`

## Notes

- This folder contains **essential infrastructure** used by the test suite
- The `example_networks.py` file is **required** for integration tests to work
- Do not remove or relocate files without updating corresponding test imports
- For research notebooks and legacy code, see `docs/examples/` and `docs/legacy/`
