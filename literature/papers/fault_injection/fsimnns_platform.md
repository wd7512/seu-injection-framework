# FsimNNs: Graph Neural Network Platform for SEU Simulation

**Lu, L., Wen, J., Krstic, M.**  
*arXiv:2511.09131, 2025*

## Summary

Open-source platform using Spatio-Temporal Graph Neural Networks (STGNNs) to accelerate SEU fault simulation.

## Key Features

1. STGNN-based acceleration using graph representations
2. Three STGNN architectures including ASPP
3. Open-source for community use

## Comparison

| Method | Speed | Accuracy |
|--------|-------|----------|
| Traditional Simulation | Slow | High |
| FsimNNs (GNN) | Fast | Approximate |
| SEU Injection Framework | Moderate | Exact |
